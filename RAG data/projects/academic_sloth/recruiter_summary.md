# Recruiter Summary — Academic Sloth

Academic Sloth is an AI-powered research assistant designed to help users interact with academic papers using conversational AI and Retrieval-Augmented Generation (RAG).

The platform allows users to:
- upload research papers
- perform semantic search
- chat with PDFs
- receive AI-generated contextual answers
- analyze academic content interactively

The project was built using a production-oriented multi-service architecture with separate frontend, backend, and AI processing layers.

# Key Technical Highlights

## Production-Style Architecture

The system follows a decoupled architecture:
- frontend layer
- Node.js orchestration backend
- Python AI service

This separation improves:
- maintainability
- scalability
- security
- independent service scaling

---

## Advanced RAG Pipeline

The project implements a dual-stage retrieval pipeline:
1. semantic vector retrieval
2. cross-encoder re-ranking

This significantly improves retrieval accuracy and reduces hallucinations.

---

## Real-Time AI Streaming

The platform streams responses using Server-Sent Events (SSE), creating a responsive conversational experience similar to modern AI products.

---

## Low-Cost AI Infrastructure

Instead of relying heavily on expensive cloud AI APIs, the system uses:
- local embeddings
- CPU-based vector generation
- ChromaDB local persistence

to reduce operational cost while maintaining strong retrieval quality.

---

## Security & Backend Engineering

The backend includes:
- JWT authentication
- OTP email verification
- protected API routing
- secure file handling
- backend proxy orchestration

The frontend never directly accesses AI infrastructure or API keys.
