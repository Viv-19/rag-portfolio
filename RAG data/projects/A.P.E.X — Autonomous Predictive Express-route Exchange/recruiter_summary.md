# Recruiter Summary — A.P.E.X (Autonomous Predictive Express-route Exchange)

## Project Overview

A.P.E.X is an AI-powered autonomous logistics intelligence platform designed to transform India's highway freight network into a real-time predictive and self-healing supply chain system.

The platform combines:
- machine learning
- graph routing algorithms
- anomaly detection
- cloud-native infrastructure
- real-time synchronization
- AI reasoning

to autonomously detect disruptions, predict cascading failures, and reroute freight traffic across logistics corridors.

The system was designed as a production-style distributed AI infrastructure platform rather than a traditional dashboard application.

# Key Technical Highlights

## FASTag-as-IoT Innovation

One of the major innovations of the project was treating:
```text
FASTag transaction events as IoT telemetry
```

instead of requiring:
- GPS hardware
- dedicated sensors
- fleet-installed devices

This allowed the platform to leverage India's existing RFID toll infrastructure for:
- freight tracking
- congestion analysis
- predictive routing intelligence

without additional hardware deployment.

---

The ML system combines:

to estimate:
- congestion probability

The models were optimized for:
- low-latency inference
- operational interpretability
- real-time deployment

---

## Autonomous Routing Engine

The platform implements:
```text
Weighted A* Search
```

using:
```text
NetworkX
```

to compute:
- disruption-aware reroutes
- low-risk paths
- congestion-aware routing

instead of relying only on shortest physical distance.

---

## Cascade Failure Modeling

The system includes:
```text
Motter-Lai inspired cascade propagation modeling
```

to simulate how disruptions:
- propagate through neighboring corridors
- increase utilization stress
- create secondary bottlenecks

This enables:
- predictive congestion analysis
- early anomaly detection
- self-healing rerouting behavior
