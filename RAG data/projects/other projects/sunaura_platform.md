# SunAura — Authorized Racold Distributor Platform

## Overview

SunAura is a premium business web platform built for:
```text
an authorized Racold water heating solutions distributor
```

The platform showcases:
- storage geysers
- tankless geysers
- heat pumps
- solar water heaters
- commercial installation projects

through a:
```text
high-performance React-based static architecture
```

- speed
- SEO visibility
- customer acquisition
- local business discoverability

The project combines:
- modern frontend engineering
- cloud deployment automation
- CDN-based scalability
- local SEO strategy

# Core Architectural Principles

The project intentionally avoids:
- traditional backend servers
- database infrastructure
- complex hosting environments

Instead, the architecture relies on:
```text
static frontend deployment
```

combined with:
- centralized configuration
- CDN caching
- serverless delivery

---

# Centralized Config System

All:
- product details
- category metadata
- business information
- contact settings

are stored in:
```text
frontend/src/data/website-config.js
```

This provides:
- simplified maintenance
- zero database costs
- centralized content management

without requiring:
- CMS infrastructure
- backend APIs
- admin dashboards

---

# Scalability Strategy

The application is deployed using:

- global edge delivery
- automatic caching

- serverless hosting

This allows the platform to handle:
```text
high concurrent traffic
```

without:
- dedicated servers

- infrastructure management

# Deployment Architecture

## CI/CD Pipeline

Deployment is fully automated using:
```text
GitHub → AWS Amplify
```

workflow integration.

---

# Deployment Flow

1. Developer pushes code to GitHub
2. GitHub triggers AWS Amplify webhook
3. Amplify executes build pipeline
4. Static assets are deployed to S3
5. CloudFront globally distributes assets
6. SSL certificates are automatically managed
